# PharmaTimer

PWA per la gestione della terapia farmacologica quotidiana.

## Stack

- React 18 + Vite
- Tailwind CSS (core utilities)
- React Router
- Dexie (IndexedDB)
- vite-plugin-pwa (service worker + manifest)
- Vitest (test logica recalc)

## Avvio

```bash
npm install
npm run dev
```

Apri http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Test

```bash
npm run test
```
